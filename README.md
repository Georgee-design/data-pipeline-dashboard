# Data Pipeline & Analysis Dashboard

A reproducible Python project that ingests raw data, cleans it with Pandas, stores it in SQLite via SQLAlchemy, and produces analysis dashboards with visualizations.

## Features
- ETL pipeline (Extract, Transform, Load)
- SQLAlchemy ORM schema
- Pandas analysis + Seaborn/Matplotlib plots
- Jupyter notebook walkthrough
- Unit tests with pytest
